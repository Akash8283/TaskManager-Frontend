# Task Manager Frontend

Frontend for the Task Manager application built using React and functional components.

## Tech Stack
- React
- JavaScript
- Tailwind CSS
- Axios

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Real-time UI updates

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Update the backend API URL if required
4. Run `npm start`
